# Data Analytics Internship Projects

This repository contains three data analytics projects completed as part of my internship. These projects demonstrate skills in data analysis, visualization, and machine learning using real-world datasets.

---

## Projects Overview

### 1️⃣ Sales Dashboard (Power BI)

**Objective:**
To analyze sales performance across different categories, regions, and time periods.

**Key Features:**

* Sales by Category
* Sales by Region
* Monthly Sales Trend
* Interactive filters (Region, Category)

**Tools Used:**

* Power BI

**Outcome:**
Built an interactive dashboard to monitor business performance and identify key sales trends.

---

### 2️⃣ HR Attrition Analysis (Python + Power BI)

**Objective:**
To analyze employee data and predict attrition (whether an employee will leave the company).

**Key Features:**

* Data cleaning and preprocessing
* Exploratory Data Analysis (EDA)
* Logistic Regression model for prediction
* Confusion Matrix and model evaluation
* Power BI dashboard for visualization

**Tools Used:**

* Python (Pandas, Seaborn, Matplotlib, Scikit-learn)
* Power BI

**Outcome:**
Identified key factors influencing employee attrition and built a predictive model to support HR decision-making.

---

### 3️⃣ Retail Business Performance Analysis (Python + Power BI)

**Objective:**
To analyze retail sales data and extract business insights related to sales, profit, and discount strategies.

**Key Features:**

* Sales by Category and Region
* Monthly Sales Trend
* Top Products by Sales
* Discount vs Profit Analysis

**Tools Used:**

* Python (Pandas, Seaborn, Matplotlib)
* Power BI

**Outcome:**
Discovered patterns in sales and profitability, helping to identify high-performing products and areas for improvement.
